# Dataset Tables

| Table | Purpose |
| --- | --- |
| `users` | Stores admin and student users |
| `missions` | Stores mission definitions |
| `mission_submissions` | Stores mission submissions and approval status |
| `point_events` | Stores point event records after mission approval |
| `educational_contents` | Stores learning content |
| `quizzes` | Stores quiz sets |
| `quiz_questions` | Stores quiz questions |
| `quiz_attempts` | Stores user quiz scores and progress |
| `badges` | Stores badge definitions and criteria |
| `user_badges` | Stores awarded badges |
| `uploaded_files` | Stores metadata for files uploaded to Blob Storage |
